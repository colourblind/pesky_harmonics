Performs frequency analysis on the specified file and outputs an image of 
the results.

Still very much WIP.

USAGE
-----

pesky_harmonics [AUDIO_FILENAME] [OUTPUT_FILENAME]

SHOULDERS STOOD ON
--------- ----- --

libav
KissFFT
libpng
